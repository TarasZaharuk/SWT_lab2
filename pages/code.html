Приклади програмного коду

На цій сторінці зібрано фрагменти коду, що використовуються в навчальних проєктах кафедри. Кожен приклад оформлено з дотриманням семантичної розмітки: блок коду — тегом <pre>, сам код — тегом <code>, а змінні та параметри — тегом <var>.

Функція обчислення суми масиву (JavaScript)

Функція приймає масив numbers і повертає суму всіх його елементів. 

function sumArray(numbers) {
  let total = 0;

  for (let i = 0; i < numbers.length; i++) {
    total += numbers[i];
  }

  return total;
}

Запит даних через Fetch API

Приклад асинхронного запиту до сервера за адресою url із наступною обробкою отриманих даних data. 

async function loadData(url) {
  const response = await fetch(url);
  const data = await response.json();

  console.log(data);
  return data;
}

Додаткові семантичні теги

П'ять тегів, не розглянутих у прикладах

Нижче наведено короткий опис і робочий приклад для кожного з п'яти семантичних тегів: <mark>, <output>, <progress>, <meter> та <wbr>.

<mark> — виділення релевантного фрагмента

Застосовується для візуального виділення тексту, який є релевантним у певному контексті, наприклад слова, знайденого в результатах пошуку.

Результат пошуку за запитом «семантика»: текст містить слово семантика у другому абзаці статті. 

<p>Текст містить слово <mark>семантика</mark> у другому абзаці.</p>

<output> — результат обчислення

Використовується для відображення результату обчислення чи дії користувача, наприклад суми, поверненої калькулятором.

5 + 7 = 12 

<form oninput="sumResult.value = parseInt(a.value) + parseInt(b.value)">
  <input type="number" id="a" value="5"> +
  <input type="number" id="b" value="7"> =
  <output name="sumResult" for="a b">12</output>
</form>

<progress> — прогрес виконання задачі

Позначає прогрес виконання задачі, що змінюється в часі, наприклад хід завантаження файлу.

Завантаження файлу: 70% 70% 

<progress value="70" max="100">70%</progress>

<meter> — скалярне значення в діапазоні

Відображає скалярне значення в межах відомого діапазону, наприклад рівень заповненості диска чи рейтинг товару.

Заповненість диска: 65% 65% 

<meter value="0.65" min="0" max="1">65%</meter>

<wbr> — можливий розрив рядка

Вказує браузеру допустиме місце для перенесення довгого слова чи URL-адреси, якщо воно не вміщується в рядок.

Довге слово: Кафедракомп’ютерногомоделюванняпроцесівісистем 

Кафедракомп&#x2019;ютерного<wbr>моделюванняпроцесів<wbr>ісистем
