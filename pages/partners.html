Партнери інституту

Кафедра співпрацює з провідними IT-компаніями регіону, які надають студентам бази для практики, менторів та запрошують випускників на стажування.

  • DevStudio

    Аутсорсингова веб-студія, партнер кафедри з практики для студентів 3–4 курсів.

    Перейти на сайт

  • CloudNet

    Провайдер хмарних рішень, що проводить спільні лабораторні з курсу хмарних технологій.

    Перейти на сайт

  • DataByte

    Компанія з аналізу даних, що пропонує теми дипломних робіт та оплачувані стажування.

    Перейти на сайт

Відгук партнера

Студенти кафедри КМПС регулярно проходять у нас практику й демонструють впевнені навички роботи з сучасним стеком веб-розробки. Керівник відділу розробки, DevStudio

Інтеграція з партнерськими сервісами

Для відображення статусу відкритих вакансій партнерів на сторінці використовується невеликий фрагмент коду, який партнер вставляє у свою систему набору:

Виклик відбувається через тег <script> з підключенням публічного API кафедри. 

fetch("https://api.kmps.hpi.edu.ua/vacancies")
  .then(response => response.json())
  .then(data => console.log(data));
